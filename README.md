Steps to run the code:

1. Clone the repository to your local machine.
    ```sh
    git clone https://github.com/aniketjhade/DZap_assigment.git
    ```

1. Install the required packages.
    ```sh
    cd Assignment
    npm install
    ```

1. Start the development server.
    ```sh
    npm start
    ```
